README

- Pour lancer le serveur backend, depuis le repertoire de travail :

Ouvrir un nouveau terminal, entrer la commande "cd backend" dans le terminal, 
puis "npm install" (pour installer les fichiers nécessaires), puis entrer la commande "nodemon server".

- Pour lancer l'appli frontend, depuis le reprtoire de travail :

Ouvrir un nouveau terminal, entrer la commande "cd frontend" dans le terminal, 
puis "npm install" (pour installer les fichiers nécessaires), puis entrer la commande "npm start".
